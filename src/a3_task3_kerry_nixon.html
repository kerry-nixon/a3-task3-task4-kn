Descriptive statistics:

To further interrogate the data, I will generate some basic summary statistics for each sample group to use for basic comparison.

# Summary statistics table using group_by() %>% summarise()
urchins_summary <- urchins_data %>% 
  group_by(common_name) %>% 
  summarise("Mean Diameter (cm)" = mean(size),
            "Median Diameter (cm)" = median(size),
            "Standard Deviation (cm)" = sd(size), 
            "Sample Size" = n())

Observations and Analysis: When reviewing the exploratory histograms, it appears that the distribution of the sample for purple urchins is positively skewed, while the sample distribution for red urchins looks to be more normally distributed. When viewing the sample distribution box plots there appears to be a fair amount of variance between the samples. The red urchin sample has more variance within the sample relative to the purple urchin sample. The difference in median diameter is 2.5 cm, which represents a sizable 43.47% difference in median diameter between the two samples. Likewise, the sample means are approximately 2.31 cm apart, representing a 41.07% difference in mean urchin diameter. This would indicate that unlikely that the samples are from the same population, and that the differences in mean and median diameters in the summary table are reflective of true differences between the urchin populations (i.e.it is possible that red urchins are larger than purple urchins). Section 2. Exploring purple urchin counts by site and year

# Creating a stored subset of purple urchin data
purple_urchins <- urchins_data %>% 
  filter(common_name == "Purple Urchin") %>% 
  mutate(date = mdy(date)) %>% 
  mutate(year = year(date)) %>% 
  mutate(site_full = case_when(
    site == "NAPL" ~ "Naples Reef", 
    site == "AQUE" ~ "Arroyo Quemado Reef", 
    site == "CARP" ~ "Carpinteria Reef", 
    site == "MOHK" ~ "Mohawk Reef", 
    site == "IVEE" ~ "Isla Vista Reef"))

# Generating a summary table containing total counts of purple urchins observed by site and year
purple_urchin_summary <- purple_urchins %>% 
  group_by(site_full, year) %>% 
  count()

Exploratory Graph of Annual Purple Urchin Counts by Site
